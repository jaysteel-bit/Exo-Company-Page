# Product Suite — The Exosystem

---

## Overview

All products are modular **namespaces** that plug into Flow OS as the foundation. They share data, agents, and institutional memory across the entire operation.

The hierarchy:
- **Exo Enterprise** = the firm / holding company
- **Flow OS** = the operating system / foundation layer
- **Namespaces** = modular departments inside the OS (Deal OS, Academy, Launch, AURA, etc.)
- **Exo AI** = the command center / AI orchestration layer
- **Steel** = future B2C consumer spinout

---

## Core Products

### Flow OS — Operations Department in a Box
**Status:** MVP prototype live (access.exoflowos.com)  
**Tagline:** "Your Unified AI Operations Layer"  
**Description:** The control layer your entire operation runs through. Every SOP, every agent handoff, every decision — logged, searchable, and owned by you. Not a dashboard you rent. A department you keep.

**Primary outcomes:**
- 20+ hours/week saved
- 50% faster onboarding
- All workflows, data sources, and decisions in one unified layer
- No more tribal knowledge walking out the door with departing employees

**Key technical concepts (use internally, not in copy):**
- Diffs architecture — every state change is logged as an immutable record
- Reflex Arc engine (SENSE → REACT → REMEMBER) for automated decision-making
- Human-in-the-loop "why" annotations — captures the reasoning behind decisions
- Namespace separation — all products share the same database schema

**Integrations:** Slack, HubSpot, Notion, Salesforce, Google Workspace — Flow OS sits on top of your existing stack.
